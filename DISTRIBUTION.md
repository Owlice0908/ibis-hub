# Ibis Hub 利用ガイド

**v0.2.75**（2026-07-02）— Windows 版

複数の Claude Code / ChatGPT (codex) セッションを 1 つのウィンドウで並列に運用できるデスクトップアプリです。

---

## インストール

1. `Ibis.Hub_0.2.75_x64-setup.exe` をダブルクリック
2. SmartScreen 警告が出たら「詳細情報」→「実行」
3. スタートメニューから **Ibis Hub** を起動

## 前提

WSL Ubuntu で以下 CLI が動くこと（Ibis Hub は WSL の `claude` / `codex` を呼び出します）:

- `claude`（未認証なら `claude setup-token`）
- `codex`（未認証なら `codex login`）

---

## 基本操作

| 操作 | 方法 |
|---|---|
| 新規セッション | 左サイドバー下部 `+ Claude` / `+ ChatGPT` / `+ Terminal` |
| タブ切替 | サイドバーのセッション名クリック、または **Ctrl+Tab** / **Ctrl+Shift+Tab** |
| レイアウト | 上部 `Single` / `Focus` / `Grid` |
| セッション並び替え | サイドバー内でドラッグ & ドロップ |
| ファイル送信 | タブヘッダー `+ File` ボタン、またはターミナルにドラッグ & ドロップ |

## 便利機能

- **画像プレビュー**（ChatGPT）: 出力の画像パスをクリック → プレビュー表示。プレビューは `＿` で最小化、履歴サムネイル表示
- **task メーター**: Claude Code の長時間コマンド走行中、タブヘッダーに 経過時間 + 過去平均 + 進捗バー
- **Ctrl+Tab で セッション切替**: ブラウザのタブ切替相当

## 通常の PC 操作との違い（重要）

ターミナル内蔵のため、通常のブラウザと **少し違う挙動** があります。

| キー / 操作 | Ibis Hub での挙動 |
|---|---|
| **Enter** | 送信（Claude/codex にメッセージ） |
| **Shift + Enter** | 改行（複数行入力） |
| **Esc** | 入力キャンセル / 応答中断 |
| **Ctrl+C** | **中断シグナル**（通常 PC の「コピー」ではない） |
| コピー | 選択で自動コピー、または **Ctrl+Shift+C** |
| ペースト | **右クリック**、または Ctrl+V |
| 全選択 | **Ctrl+Shift+A**（Ctrl+A は行頭移動） |
| **Ctrl+F** | セッション内 文字列検索 |
| **Alt+クリック** | ターミナル上の任意位置にカーソル移動 |
| **Shift+マウスホイール** | 履歴を上下スクロール |
| ファイル選択・入力 | Tab で移動できません。マウスクリックで選択 |
| 履歴を遡る（Claude）| **Shift+ホイール**（xterm レベルの scrollbar は Alt Screen Buffer の制約で動きません） |

## 困った時

| 症状 | 対処 |
|---|---|
| 画面が黒い | レイアウトを切替（Grid → Single 等）で復旧するケース |
| 応答が来ない | Esc を 1-2 回押して中断 → 再入力 |
| 重い | 開きすぎのタブを `✕` で閉じる |
| 認証切れ | WSL で `claude setup-token` / `codex login` を再実行 |
| その他 | **nakamura さんまで** |
