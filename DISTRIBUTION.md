# Ibis Hub — 利用ガイド (Windows 版)

## これは何か

複数の Claude Code / ChatGPT (codex) セッションを 1 つのウィンドウで並列に運用できるデスクトップアプリです。作業ごとにタブを分けて、レイアウト（Single / Focus / Grid）を切り替えながら同時進行できます。

---

## インストール

1. `Ibis.Hub_0.2.52_x64-setup.exe` をダブルクリック
2. SmartScreen 警告が出たら「詳細情報」→「実行」
3. インストール完了後、スタートメニューから **Ibis Hub** を起動

---

## 前提: WSL + Claude / codex CLI

Ibis Hub 内のセッションは WSL Ubuntu 上の `claude` / `codex` CLI を呼び出します。事前に以下が動く状態にしてください。

- `claude` コマンドが WSL で動くこと（未認証なら WSL で `claude setup-token`）
- `codex` コマンドが WSL で動くこと（未認証なら WSL で `codex login`）

---

## 使い方の要点

- **新規セッション**: 左サイドバー下のボタン `+ Claude` / `+ ChatGPT` / `+ Terminal`
- **タブ切替**: サイドバーのセッション名クリック
- **レイアウト切替**: 上部の `Single / Focus / Grid`
- **画像プレビュー**（ChatGPT セッション）: 出力に画像パス（`~/.codex/generated_images/...` 等）が表示されたらクリック → 画面上でプレビュー
  - プレビュー右上の `_` で **最小化**（右下バッジに畳む）、`✕` で **閉じる**
  - 何枚も生成すると **セッション内の履歴サムネイル** が下に並び、クリックで切替可能
- **background タスクの進捗**: 長時間コマンド（build 監視等）を走らせるとタブヘッダーに **経過時間 + 進捗バー + 過去平均** が表示されます
- **セッション並び替え**: サイドバー内でセッションをドラッグ & ドロップ

## 操作ガイド（通常の PC 操作と違うところ）

ターミナル（xterm）を内蔵しているため、通常のブラウザやアプリと **少し違う挙動** があります。以下は代表的な差分です。

### 文字選択・コピー・ペースト

| やりたいこと | 操作 |
|---|---|
| 文字を選択したい | 通常はマウスドラッグ。**Claude や codex が対話モードでマウスを掴んでいる時は Shift を押しながらドラッグ** |
| コピー | 選択すれば **自動でクリップボードにコピー** されます（Ctrl+C は使いません — 下記参照） |
| ペースト | **右クリック** で PTY にペースト、または通常の Ctrl+V |
| 全選択（Ctrl+A） | ターミナル内での Ctrl+A は **行頭にカーソル移動**（Bash/readline の仕様）。ブラウザ的な「全文選択」ではありません |
| ⚠️ Ctrl+C | ターミナルでは **中断シグナル送信** です。コピー用途では使いません |

### キーボード

| キー | 挙動 |
|---|---|
| Enter | **送信**（Claude/codex にメッセージを送る） |
| Shift+Enter | **改行**（複数行入力） |
| Esc | 入力中は入力キャンセル、応答生成中は **応答中断** |
| Ctrl+F | セッション内で **文字列検索** |
| Ctrl+Z | ターミナルでは **プロセス suspend**（元に戻すではありません） |
| ⚠️ フォーカス | Tab キーではタブを移動できません。**操作したいセッションを一度クリック** してからキー入力してください |

### マウス

| 操作 | 挙動 |
|---|---|
| 左クリック | フォーカスを当てる |
| **Alt+クリック** | ターミナル上の任意位置に **カーソル移動**（長い入力の修正が楽） |
| 右クリック | ペースト |
| ダブルクリック | 単語選択（通常と同じ） |
| マウスホイール | ターミナル内スクロール（scrollback） |
| **Shift+ホイール** | scrollback の上下（通常のブラウザだと水平スクロールですが、こちらは縦スクロール優先） |
| ドラッグ & ドロップ | ファイルやフォルダをターミナルにドロップすると **パスが PTY に送信** されます |

### 独自機能

| 機能 | 説明 |
|---|---|
| `+ File` ボタン | タブヘッダーのボタン。ファイル / フォルダを選択して、そのパスを PTY に送信できます |
| 画像パスのクリック | ChatGPT が生成した画像のパスが出力に流れると、**クリックでプレビュー表示** |
| 右上の task メーター | Claude Code の Bash background コマンドを走らせると **経過 / 過去平均 / 進捗バー** で残り時間の目安が表示 |
| Sidebar ドラッグ | セッション行を **ドラッグ & ドロップで並び替え** できます |
| スクロールバー | ターミナル右端に **常時表示** されるスクロールバーで scrollback を上下できます |

### 表示

- **等幅フォント固定**（PlemolJP Console）: 日本語含めて綺麗に揃うようにこのフォントを使っています
- **絵文字・特殊記号**: 幅の計算がずれる場合があります（xterm の仕様上、完全対応は困難）

## 困った時

| 症状 | 対処 |
|---|---|
| 画面が黒い | レイアウトを別モードに切り替え（Grid → Single 等）で復旧するケースがあります |
| 応答が来ない | Esc を 1-2 回押して現在の入力・応答を中断してから再入力 |
| 全体的に重い | 開いているセッション数が多すぎる可能性。使わないタブは `✕` で閉じる |
| 何か根本的におかしい | Windows の **タスクマネージャで Ibis Hub を終了** → 再起動 |
| 認証が切れた | WSL で `claude setup-token` / `codex login` を再実行 |
| その他 | **nakamura さんまで** |
